🌦️ Weather Analytics Dashboard (Power BI)

A modern and interactive **Weather Analytics Dashboard** built using **Microsoft Power BI** and real-time weather API data. This dashboard provides comprehensive insights into weather conditions, forecasts, and air quality metrics in a visually appealing and user-friendly interface.

## 📌 Overview

This project showcases the integration of **real-time weather data** into a dynamic Power BI dashboard. It enables users to monitor current weather conditions, analyze trends, and understand environmental factors such as air quality and precipitation.

## 🚀 Features

🌍 Real-time weather data visualization
🌡️ Current temperature and condition display
 📊 7-day weather forecast with trend analysis
💨 Air Quality Index (AQI) monitoring
🌫️ Pollutant level breakdown (PM10, PM2.5, CO, NO2, SO2, O3)
🌅 Sunrise and sunset timing
💧 Humidity, wind speed, pressure, and visibility insights
🌧️ Chances of rain visualization
🎯 Clean and modern UI design (dark theme)

## 🛠️ Tech Stack

* Microsoft Power BI
* Weather API (for real-time data)
* Power Query (data transformation)
* DAX (data analysis expressions)

## ⚙️ How It Works

1. Weather data is fetched using a public Weather API
2. Data is cleaned and transformed using Power Query
3. Calculations and measures are created using DAX
4. Data is visualized using interactive charts and cards in Power BI
